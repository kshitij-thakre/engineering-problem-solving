# System Design & Scalable Architectures

This module covers architectural patterns, caching strategies, token security, and session management.

---

## 📂 Module Directory Index

* **[LRU Cache](./lru_cache.md)**: HashMap + Doubly Linked List implementation for $O(1)$ Least Recently Used cache updates.
* **[LFU Cache](./lfu_cache.md)**: Dual HashMap + Frequency bucket implementation for $O(1)$ Least Frequently Used caches.
* **[Cache Eviction Flow](./cache_eviction_flow.md)**: Sequence and workflow diagrams tracing cache updates and evictions.
* **[Secure Session Management](./secure_session_management.md)**: JWT authorization loops, Token Refresh Rotation, Keychain/Keystore hardware-enclave protections, and OAuth 2.0 PKCE.
